# Smart Storage Management – Navtej Technologies

## Installation & Setup

Follow these steps to install and run the program on your computer:

1. **Download the project**
   - Download the project as a `.zip` file.

2. **Extract the files**
   - Extract the `.zip` file to your `Documents` folder (or any folder you prefer).

3. **Locate the main file**
   - Open the extracted folder.
   - Find the file named `index.html`.

4. **Create a desktop shortcut**
   - Right-click on `index.html`.
   - Click on **Create shortcut**.  
     - If you do not see this option directly, click **Show more options** and then choose **Create shortcut**.
   - A shortcut to `index.html` will be created inside the same folder.

5. **Move the shortcut to Desktop**
   - Cut or copy the newly created shortcut.
   - Paste it into your **Desktop** folder.

6. **Run the program**
   - Whenever you want to use the Smart Storage system:
     - Go to your desktop (home screen).
     - Double-click the `index.html` shortcut.
   - The application will open in your default web browser.
